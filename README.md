# read Programming in OCaml

本「プログラミング in OCaml ～関数型プログラミングの基礎からGUI構築まで～」の練習問題

## How

Install OCaml & rlwrap in vagrant and startup with `rlwrap ocaml`.

## Enhancement

```sh
vim ~/.bashrc
# add following row
alias rocaml='rlwrap ocaml'
# Start up
rocaml
```

## Run file

In OCaml:

`#use "exchange.ml";;`

## Quit

`#quit;;` or `control-D`
